# Sistem Informasi Absensi Guru

Project ini adalah aplikasi web absensi guru untuk `MI AL-KHAIRIYAH BADAMUSALAM` berbasis `Flask + SQLite` dengan UI `Bootstrap 5`.

Fitur utama:
- Login multi-role `Admin` dan `Guru`
- Dashboard statistik admin dan guru
- CRUD data guru
- Upload foto guru
- Absensi harian `check-in` dan `check-out`
- Perhitungan keterlambatan otomatis
- Validasi absensi
- Rekap harian, mingguan, bulanan, semester, tahunan
- Export laporan PDF dan Excel
- Pengaturan profil madrasah, logo, dan tanda tangan
